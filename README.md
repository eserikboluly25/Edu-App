✨ Edu App — Learning Web App
This is a small learning web app I built using Streamlit.
The goal was to practice Python and create something useful for studying — like vocabulary, basic math, and simple task planning.
I also used tools like ChatGPT and Gemini in some parts (mainly for ideas, debugging, and UI tweaks), but the project itself I assembled and understood step by step.

🚀 What it does
📚 Flashcards
Shows random words
You can reveal the translation

🧠 Mental Math
Generates simple math problems
Lets you check your answer
You can get a new task anytime

📖 Vocabulary
Add your own words and translations
Saves them in a JSON file
Shows the list with timestamps

📅 Planner
Add tasks
Saves them locally
Shows progress
Option to clear everything
🛠️ Built with
Python
Streamlit
JSON (no database, just simple storage)






📌 Notes

This is a beginner project, but it helped me understand:

how Streamlit works
how to manage simple app state
how to store and load data

It’s not perfect, but it works and can be improved further.
